Element Description Example
<!DOCTYPE html> Defines the document type and HTML version. <!DOCTYPE html> <html> <head> <title>Document Title</title> </head> <body> Document body here </body> </html>
<a href="path"> Creates a hyperlink to another page. <a href="https://www.ibm.com">IBM</a>
<article> A self-contained piece of content. <article class="CSS-Style-Reference"> <h2>HTML</h2> <p>HTML stands for Hypertext Markup Language.</p> </article>
<aside> Content related to the main content. <aside> Use the aside tag for tangential content. </aside>
<body> Contains visible page content. <body> Document body here </body>
<div> Generic block-level container for styling. <div> Content inside a div </div>
<form> + <fieldset> Form for user input, with grouped fields. <form action="/submit"> <fieldset> <legend>User Info</legend> <input type="text" name="name"> </fieldset> </form>
<figure> & <figcaption> Image with caption. <figure> <img src="img.jpg"> <figcaption>Caption text</figcaption> </figure>
<header> Introductory section of a page. <header> <h1>Welcome</h1> </header>
<img> Displays an image on the page. <img src="image.jpg" width="300" height="200">
<input> Used for user data entry. <input type="text" name="username">
<link> Link an external CSS file. <link rel="stylesheet" href="styles.css">
<nav> Container for navigation links. <nav> <a href=\"index.html\">Home</a> </nav>
<ol> & <ul> Ordered and unordered lists. <ul><li>Item 1</li></ul> <ol><li>First</li></ol>
<script> Embeds or references JavaScript. <script> alert('Hello'); </script>
<style> Internal CSS definitions. <style> p { color: red; } </style>
<table> Defines a table structure. <table> <tr><th>H1</th><th>H2</th></tr> <tr><td>Data</td><td>Data</td></tr> </table>
